Hacon CSS Library


This library provides a set of CSS classes to standardize and optimize typical layouting, typography and styling tasks.
The proposed SCSS structure offers a simple way to extend the library for your own needs.

  • Responcivity
  • Layouting

Responsivity

Responcivity works based on breakpoint prefixes which are defined in $breakpoints variable located in variables.scss

Initial Values

Large Desktop Width -ldw max-width: 1600px
Small Desktop Width -sdw max-width: 1200px
Tablet Width -tw max-width: 992px
Large Mobile Width -lmw max-width: 768px
Mobile Width -mw max-width: 576px

Classes Template

property-breakpoint-value

property is the CSS property you want to change, breakpoint is the breakpoint prefix and value is the value you want to set.

Examples: fs-mw-2-25 sets the font-size to 2.25rem on mobile devices. fs-sdw-2-25 sets the font-size to 2.25rem on small desktop devices.

Flex

  • flex{-breakpoint}-row creates flex row.
  • flex{-breakpoint}-column creates flex column.
  • flex{-breakpoint}-row-reverse creates flex row with reverse order.
  • flex{-breakpoint}-no-wrap disbales wrapping.
  • flex{-breakpoint}-col auto fills free space in a row.
  • flex{-breakpoint}-col-{from 1 to 12} sets the width of the column to the specified fraction of the row.
  • flex{-breakpoint}-reset-order sets the order to unset.
  • flex{-breakpoint}-first-col sets the order to -1.
  • flex{-breakpoint}-last-col sets the order to 1.

Grid

  • grid creates grid layout.
  • grid{-breakpoint}-col-{from 1 to 12} sets the width of the column to the specified fraction of the row.
  • grid{-breakpoint}-col-start-{from 1 to 12} sets the start position of the column.
  • grid{-breakpoint}-col-end-{from 1 to 13} sets the end position of the column.
